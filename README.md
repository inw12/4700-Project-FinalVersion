# CS 4750 Final Project
### Title: MOVE!
### Author: Ian Wong

## Description:
A top-down 2D action RPG.

# Installation
1. Install the .zip file & extract
2. In the folder, run MOVE!.exe to start the game

# Controls
- **WASD**: Movement
- **Left Click**: Attack
- **E**: Interact
- **Space**: Dodge
- **Esc**: Pause/Unpause

## Debugging
- If the game ever freezes up while playing, terminate the program by pressing:
  - ALT+F4 (Windows)
  - CMD+OPTION+ESC (Mac)
